建設現場では、切り傷・打撲・骨折・やけどなど、様々な事故が起こる可能性があります。医療機関に到着するまでの応急処置が、その後の経過を大きく左右することもあります。この記事では、建設現場で知っておくべき応急処置の基本手順を解説します。

応急処置の基本原則

応急処置を行う際の基本原則は以下のとおりです。

  • 自分の安全確保：二次災害を防ぐ
  • 状況の観察：怪我の程度を判断
  • 適切な処置：自分ができる範囲で
  • 医療機関への連絡：必要に応じて119番
  • 冷静な対応：パニックにならない

切り傷の応急処置

建設現場で最も多い切り傷への対応を紹介します。

  1. 清潔な布・ガーゼで圧迫止血
  2. 患部を心臓より高く上げる
  3. 水で洗い流す（表面の汚れ除去）
  4. 消毒液で消毒
  5. 清潔な絆創膏・ガーゼで保護
  6. 深い場合は医療機関へ

出血の程度別対応

出血の程度によって対応が変わります。

程度対応
少量圧迫止血・絆創膏
中等度圧迫止血・病院へ
大量即119番・止血継続
動脈性緊急性最優先・119番

打撲・捻挫の応急処置

打撲・捻挫には「RICE処置」が基本です。

  • R（Rest）：安静にする
  • I（Ice）：氷で冷やす（15〜20分）
  • C（Compression）：包帯で圧迫
  • E（Elevation）：患部を高く上げる

骨折の応急処置

骨折が疑われる場合の対応を紹介します。

  1. 患部を動かさない
  2. 添え木・固定材で固定
  3. 患部を冷やす
  4. 119番通報
  5. ショック症状に注意
  6. 無理に移動させない

やけどの応急処置

やけどは早い対応で傷の深さが変わります。

  • すぐに流水で冷やす（15〜20分）
  • 衣服は無理に脱がさない
  • 水ぶくれを破らない
  • 清潔な布で保護
  • 広範囲・深い場合は病院へ
  • 電気やけどは感電対策を優先

熱中症の応急処置

夏場の建設現場では熱中症への対応も重要です。

  1. 涼しい場所に移動
  2. 衣服を緩める
  3. 体を冷やす（首・脇・足の付け根）
  4. 意識があれば経口補水液を飲ませる
  5. 意識がない場合は119番
  6. 回復まで付き添う

心肺蘇生法

心肺停止が疑われる場合の対応を紹介します。

  • 意識・呼吸の確認
  • 119番通報
  • 胸骨圧迫（30回）
  • 人工呼吸（2回）
  • 繰り返し
  • AEDの使用

感電事故の応急処置

感電事故では、二次災害に最大の注意が必要です。

  • 電源を遮断（ブレーカーを落とす）
  • 乾いた木の棒等で電線を払う
  • 直接触らない
  • 心肺蘇生の準備
  • やけどの処置
  • 119番通報

救急箱の整備

建設現場には救急箱を常備することが重要です。主な中身は以下のとおりです。

  • 絆創膏・ガーゼ・包帯
  • 消毒液
  • 三角巾
  • ハサミ・ピンセット
  • 冷却シート・冷湿布
  • 経口補水液
  • テープ類

応急処置講習の受講

応急処置を正しく行うには、事前の講習受講が有効です。

  • 消防署の救命講習
  • 赤十字の応急手当法講習
  • 会社の安全衛生教育
  • AED講習

まとめ

建設現場での応急処置は、命を救い、治療を早める重要なスキルです。事前に知識を身につけておくことで、万が一の際に適切な対応ができます。救急講習を受講し、現場の救急箱を整備して、安全な職場を作りましょう。
